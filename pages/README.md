# Pages

Thư mục này chứa các Vue components để tạo file-based routing.

Nuxt tự động tạo routes dựa trên cấu trúc file.

**Ví dụ:**
- `pages/index.vue` → `/`
- `pages/about.vue` → `/about`
- `pages/users/[id].vue` → `/users/:id`
- `pages/products/[...slug].vue` → `/products/*`

Xem thêm: https://nuxt.com/docs/guide/directory-structure/pages
